# 스프링 JPA 기반 웹 프로젝트 샘플링

이프로젝트는 인프런 백기선님의 [스프링과 JPA 기반 웹 애플리케이션 개발](https://www.inflearn.com/course/%EC%8A%A4%ED%94%84%EB%A7%81-JPA-%EC%9B%B9%EC%95%B1/dashboard) 을 공부하며 정리한 프로젝트 입니다.
